Веб-страница содержит содержит в себе: описание разделов выбора операционной системы, выбора файловой системы, 3 средств установки и удаления программного обеспечения, 3 средств создания, редактирования и обслуживания виртуальных носителей.

Для навигации по странице используется выезжающее меню, открыть которое можно по кнопке в верхнем левом углу.

С каждой страницы веб-страницы можно вернуться на главную с помощью выезжающего меню или закрепленного горизонтального.
